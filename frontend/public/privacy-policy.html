Privacy Policy for Kity

Last Updated: December 2, 2025

TL;DR: Kity adds keyboard shortcuts to ChatGPT. All code runs locally in your browser. We do not collect, transmit, or sell any personal data. Optional support payments happen on Stripe's hosted checkout page; Kity never sees payment details.

1. Introduction

Kity ("we", "our") is a browser extension that provides site-aware keyboard navigation for ChatGPT (chatgpt.com and chat.openai.com). This policy explains how we handle information when you use the extension.

2. What We Do Not Collect

  • No personal information is collected, stored, or transmitted.
  • No browsing history, page content, or chats are sent to our servers (we do not operate servers for Kity).
  • No analytics, tracking pixels, or advertising SDKs are included.
  • No remote code is fetched or executed; all code ships inside the extension package.

3. Information We Store Locally

Stored only in your browser via chrome.storage.sync:

  • Extension on/off state
  • Theme preferences

Nothing is sent to any external server.

4. Optional Support Payments

If you choose to support Kity, you are redirected to Stripe's hosted checkout page. Payment data stays with Stripe. Kity does not collect, process, or store payment details or personal information.

5. Browser Permissions Explained

  • clipboardWrite: Copy selected text to your clipboard when you trigger the copy shortcut.
  • storage: Save your Kity preferences locally in Chrome sync storage.
  • Host permissions (chatgpt.com, chat.openai.com): Inject the keyboard navigation script only on ChatGPT so shortcuts and scrolling work.
  • No "tabs" permission: Commands are limited to ChatGPT tabs using host-scoped lookups; broad tab access is not requested.

Remote code: Kity does not fetch or execute remote code. Everything is packaged with the extension.

6. Data Security

  • No user data is transmitted to us.
  • Optional Stripe checkout uses HTTPS and is handled directly by Stripe.
  • Extension code is readable and unobfuscated.

7. Children's Privacy

Kity is not directed to children under 13. We do not knowingly collect personal information from children.

8. Your Choices

  • Uninstall the extension to remove all local data.
  • Clear Chrome extension data to reset settings.

9. Changes to This Policy

We may update this policy and will revise the "Last Updated" date above.
