Keyboard shortcuts for ChatGPT
Keyboard shortcuts are ready for ChatGPT.
  • Toggle pane focus: Ctrl+Left / Ctrl+Right
  • Smooth scrolling: Ctrl+Up / Ctrl+Down
  • Copy selection: Ctrl+C
  • Navigate user messages: Ctrl+Shift+Up / Ctrl+Shift+Down
  • Jump to first/last message: Ctrl+A / Ctrl+Z
  • Open folder/chats: Ctrl+Enter
Kity runs locally and is free to use.
